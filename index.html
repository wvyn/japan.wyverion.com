> Japanese Particles

Particle Meaning / Function Direct Translation Example Sentence Direct Translation of Sentence
は Topic marker (topic marker) わたしは学生です。 As for me, (I) am student.
が Subject marker / emphasis (subject marker) 猫が好きです。 Cat (subject) like (I).
を Direct object marker (object marker) 本を読みます。 Book (object) read (I).
に Indicates destination/time/purpose to / at / in / on 学校に行きます。 School (to) go (I).
で Indicates location of action at / in / by 公園で遊びます。 Park (at) play (I).
の Possessive / descriptive ’s / of 私の本です。 My book (is).
と And / with and / with 友達と話します。 Friend (with) talk (I).
も Also / too also / too 私も行きます。 Me also go.
から From / because from / because 学校から帰ります。 School (from) return (I).
まで Until / as far as until / as far as 駅まで歩きます。 Station (until) walk (I).
へ Indicates direction (similar to に but emphasizes direction) towards 日本へ行きます。 (I) go to Japan.
か Question marker ? これは何ですか。 This is what?
ね Seeks agreement / confirmation right? / isn’t it? いい天気ですね。 Nice weather, isn’t it?
よ Emphasis / assertion you know / for sure これは大事ですよ。 This is important, you know.
や Lists examples (non-exhaustive) and / such as 本やノートがあります。 There are books and notebooks (among other things).
ぞ Emphasis, masculine / rough speech (assertive ending) 行くぞ！ (I’m) going!
ぜ Emphasis, masculine casual (assertive casual) 楽しいぜ。 (It’s) fun, man.
かな Indicates uncertainty / wonder I wonder 明日雨かな。 I wonder if it will rain tomorrow.
しか “Only” (with negative verb) only 100円しかない。 (I) have only 100 yen.
でも “But” / “Even” but / even 安くても買いません。 Even if it’s cheap, I won’t buy it.
